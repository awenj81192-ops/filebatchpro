FileBatch Pro v1.0 - 文件批量处理工具箱
========================================

【功能介绍】
一键搞定日常文件处理三大痛点：

1. 批量重命名
   - 添加前缀/后缀
   - 查找替换文件名中的文字
   - 按序号批量重命名（支持自定义起始编号）
   - 支持按文件类型筛选

2. 图片批量压缩
   - 自定义压缩质量（1-100）
   - 支持限制最大宽度
   - JPEG/PNG/BMP/WEBP 全格式支持
   - 保持原图比例，智能缩放
   - 平均压缩50%-80%，肉眼无损

3. 文件归类整理
   - 按文件类型自动归类（图片/文档/视频/音频/压缩包/代码）
   - 按年份归类
   - 按月份归类
   - 自动处理重名冲突

【使用方法】
1. 双击运行 FileBatchPro.exe
2. 选择要处理的功能标签页
3. 选择文件夹 -> 设置规则 -> 执行

【系统要求】
Windows 7/8/10/11
不需要安装Python或其他运行环境

【常见问题】
Q: 会不会损坏原文件？
A: 重命名和归类仅改变文件名/位置，不修改文件内容。
   图片压缩会创建新文件到输出文件夹，原图不受影响。

Q: 支持批量处理多少文件？
A: 支持任意数量文件，已测试10000+文件无压力。

Q: 压缩后图片变模糊怎么办？
A: 调高"质量"参数（推荐75-85），或取消"最大宽度"限制。
