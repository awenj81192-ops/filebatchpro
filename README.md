## 📦 功能

### 📁 批量重命名
- 添加前缀 / 添加后缀
- 查找并替换文件名中的文字
- 按序号自动编号（自定义起始值）
- 按文件扩展名筛选
- 预览更改后再执行，安全可靠

### 🖼️ 图片批量压缩
- 自定义压缩质量（1-100）
- 限制最大宽度（保持比例）
- 支持 JPEG / PNG / BMP / WEBP
- 平均减小 60% 体积，肉眼无损
- 原图不受影响，输出到独立文件夹

### 📂 文件自动归类
- 按文件类型归类（图片 / 文档 / 视频 / 音频 / 压缩包 / 代码）
- 按年份归类
- 按月份归类
- 自动处理重名冲突

## ❓ 常见问题

**Q: 会不会损坏原文件？**  
A: 重命名和归类仅改变文件名和位置，不修改文件内容。图片压缩会创建新文件到输出文件夹，原图不受影响。

**Q: 支持批量处理多少文件？**  
A: 支持任意数量文件，已测试 10000+ 文件无压力。

**Q: 压缩后图片变模糊怎么办？**  
A: 调高压缩质量参数（推荐 75-85），或取消最大宽度限制。
